## Overview
This project converts the Airbnb database system into a REST API
using Flask. It allows users and bookings to be managed via HTTP requests.

---

## API Endpoints

| Method | Endpoint | Description |
|-------|--------|-------------|
POST | /add_user | Add new user  
POST | /add_listing | Add new listing  
POST | /book | Book a property  
GET | /bookings | View all bookings
